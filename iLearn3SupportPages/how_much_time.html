How Much Time?

Not everyone will need the same amount of time to be successful in this class. Some will need more time than others. This tool is designed to help you figure out how much time you will need to reserve out of your busy weeks for this class.

To use it, you will first need to honestly self-assess your preparation for CIT 360. Read the descriptions for the preparation levels below. In order to claim preparation at any level, the statements for all lower levels must also be true. For example, if you claim a preparation level of 4 the descriptions for levels 3,2,and 1 must also be true. It is vital to your ability to mangage your time that you select your true level of preperation.

Your instructor will never ask for nor see your preparedness selections. Do not try to deceive yourself. It is better to select something that feels a little too low than to stretch your claim to a higher level.

Preparedness Self-Assessment Levels

  1. I can read code
  2. I can write code for loops, arrays, and methods when given line-by-line guidance
  3. I can recite what an Instance is and what a class is for
  4. I can imagine and correctly implement Java class code to represent concrete types of objects such as dogs, trees, etc. and correctly use local variables, parameters, and class attributes
  5. I can correctly implement multi-level inheritance of non-concrete objects, such as an Idea class, using Java classes and correctly apply super-classes and interfaces
  6. I can find a real-life need others have, imagine and create an application design, and then write the code for the full application

Now that you have honestly selected your level of preperation for CIT 360, decide what grade you want to achieve. Find the grade you want to achieve in the first column and your preparation level in the first row of table below. The intersection of the grade row with the preparation level column contains the approximate weekly hours you need to reserve in your schedule to achieve your grade goal for CIT 360.

If you are in a face-to-face class this number of hours includes in-class and out-of-class time. If you are in an online class it shows you the total time you will need to dedicate out of your weekly schedule. These numbers are only approximations. They assume you are using the hours at a high degree of efficiency.

While these numbers are approximate, they should give you an idea of the time commitment you will need to reach your desired level of success given your level of preparedness.
